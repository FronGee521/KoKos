# KoKos
Your local Cookbook

**KoKos** — это прогрессивное веб-приложение (PWA) для управления рецептами, планирования меню и создания списков покупок. Работает полностью в браузере, данные хранятся в localStorage.

**Демо:** [откройте index.html в браузере]

## Возможности

### Управление рецептами
- Создание, редактирование и удаление рецептов
- Добавление фото из галереи (конвертируется в base64)
- Категории: завтраки, супы, салаты, горячее, выпечка, десерты, напитки, закуски
- Уровни сложности: легко, средне, сложно
- Поиск по названию, ингредиентам и категориям
- Избранное (звёздочка)
- Фильтр "только избранное"

### Планирование
- Планировщик на неделю (ПН-ВС)
- Список покупок из запланированных рецептов
- Счётчик добавлений в план

### Инструменты
- Кухонный таймер (с установкой минут/секунд)
- Конвертер мер (г, мл, стаканы, ст/ч ложки, штуки)

### Интерфейс
- Светлая/тёмная тема (ручной выбор + авто)
- Русский/английский язык
- Адаптивный дизайн под мобильные устройства (max-width: 480px)
- Стильные диалоги вместо alert/confirm/prompt
- Toast-уведомления

### Синхронизация
- Экспорт/импорт через QR-код (api.qrserver.com)
- Все данные в localStorage

### PWA
- Установка на телефон как приложение (через браузер)
- Service Worker для офлайн-доступа (настраивается отдельно)

## Установка

1. Скачайте файл `index.html`
2. Откройте в любом современном браузере
3. Для установки на телефон: меню браузера → "Установить приложение"

## Технологии

- Один HTML-файл (HTML + CSS + JavaScript)
- LocalStorage для хранения данных
- QR Server API для QR-кодов
- Шрифты Google Fonts (Montserrat, Russo One)

## Цветовая схема

- Основной цвет: #1A9170
- Светлый фон: #F5FAF8
- Тёмный фон: #121E1A
